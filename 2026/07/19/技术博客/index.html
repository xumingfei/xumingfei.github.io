今天开通了hexo博客,希望能持续输出相关的一些知识和技术沉淀

一、配置github当做博客网站快速搭建步骤（Hexo 最简教程，国内推荐）

1. 前置工具

安装 Git、Node.js（LTS 版本）

2. 创建 GitHub 专属仓库

  1. GitHub 右上角 + New repository
  2. 仓库名必须严格：你的GitHub用户名.github.io（大小写一致）
  3. 勾选 Public，创建仓库

3. 本地初始化 Hexo

4. 配置一键部署到 GitHub

  1. 安装部署插件,修改 _config.yml deploy 配置
  1. 发布博客命令

5. 写文章、换主题

文章存放在 source/_posts，Markdown 格式；下载主题放入 themes，修改 _config.yml 的 theme: 主题名 即可切换样式。

6.部署

常见问题

1.本地没安装git和node

  • 需要先安装 Git 并配置用户名邮箱：
  • npm下载慢/报错

切换淘宝镜像再安装

2. 出现443没有权限,这是很正常的一个问题,建议改为ssh方式(免密码推送)
